## 开发实现

### 开发原则

基于 ENFI-CLI 开发。

#### 前端技术

PC：**Vue2.x + Antd1.X + Js**

H5：**Vue2.x + Vant2.X + Js**

#### 后端技术

**SpringCloud + Nacos + Java**

### 开发框架获取

``` sh
# 环境
node >= 16.X
# 安装
npm i -g enfi-cli
# 更新
npm update -g enfi-cli
# 查询
enfi
# 创建项目example
enfi create example
# 使用问腿
参考项目根目录【README.md】或【USAGE.md】
```
